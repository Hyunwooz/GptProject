# OrmiProject

-   ChatGPT를 이용한 개인 프로젝트 Repo입니다.

## 1. 목표와 기능

### 1.1 목표

-   ChatGPT를 이용하여 업무의 생산성을 높일 수 있는 서비스

### 1.2 기능

-   chatGPT API 를 이용해서 광고 스크립트를 자동 생성
-   광고의 유형별 다른 템플릿으로 답변 제공
